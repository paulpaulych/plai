# Версия 1 (Однопользовательский арканоид)

## Правила игры

Время в игре дискретное.
В промежутке времени между изменениями состояния игры, Игрок, может зарегистрировать свой ход.
При изменении состяния, Игра производит ходы, зарегистрированные игроком, а затем рассчитывает новое состояние.
